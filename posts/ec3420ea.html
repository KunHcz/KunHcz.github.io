程序设计基础第六章习题

——kunhcz——

1.

知识点：

  • 指针
  • 冒泡排序

程序分析：使用了swap函数进行交换操作（就是简单的冒泡排序），在main函数里这三个整数通过指针的形式传递给函数，这意味着swap函数内部对这些值的任何修改都会影响到函数外部的原始变量。

2.

知识点：

  • 指针
  • 冒泡排序

程序分析：与第1题类似，也是使用冒泡排序。需要注意的是在C++中，数组名本身就是一个指向数组第一个元素的指针。所以，当你传递一个数组到一个函数时，实际上是传递了一个指向数组的指针，而不是整个数组的拷贝。因此，在将数组传入sort函数时，你不需要使用&运算符。

3.

知识点：

  • 指针

程序分析：主要使用三个函数实现功能，input函数输入十个整数，swapnums函数寻找最小最大值的索引并与第一个和最后一个数对换（注意最大值和最小值在两端的情况），output函数输出对换后的数组。

4.

知识点：

  • 指针
  • 引用
  • 向量数组

程序分析：主要功能实现在于change函数：先进行了一个外部循环，循环次数为m，也就是需要移动的位置数。在每次循环中，首先将数组的最后一个元素保存在临时变量temp中，然后进行一个内部循环，从数组的最后一个元素开始，将每个元素替换为它前面的元素，直到到达数组的第二个元素。这样，所有的元素都向后移动了一个位置。然后，将temp中保存的原始最后一个元素放到数组的第一个位置。这样，就完成了一次所有元素向后移动一个位置的操作。重复这个过程m次，就可以实现所有元素向后移动m个位置。

5.

知识点：

  • 迭代器：是一种允许遍历容器（如向量）元素的对象。迭代器提供了访问和操作容器元素的方式。
  • 向量数组：可以动态调整数组大小，而不需要提前确定数组大小。

程序分析：先创建一个向量数组arr，vector<int>::iterator p声明了一个专门用于遍历整数向量元素的迭代器p。 arr.push_back(i+1);为每个人编号，从1至n。m为退出人数，当退出人数比n-1少时（即未退出人数大于1时）执行循环体。p = num.begin() + i;将迭代器 p 指向 arr（整数向量）的第 i 个位置，若k=3，将当前位置的值置为0，表示这个人退出。if (i == n) i = 0;如果移动到末尾，将 i 恢复为0，重新从开头开始报数。最后找出未被编号为0的人，即为最后一个人。该题源于约瑟夫问题，难度较大。

6.

知识点：

  • 引用
  • 传递

程序分析：在void getStrLength(const string &str, int &length)中第一个形参是常量引用，比第二个多了const，其作用在于避免拷贝开销：字符串通常是较大的数据结构，如果直接传递字符串而不是引用，会触发一次拷贝操作，造成额外的内存和性能开销。使用引用可以避免这种拷贝，提高效率。而我们需要在函数里修改length的值，故不需const。

7.

知识点：

  • 字符串处理

程序分析：该题注意点在于substr函数用于获取从第 m 个字符开始的子字符串。如果m被设置为5，那么source.substr(m) 将返回从第6个字符（即空格）开始到字符串末尾的所有字符。故由题意设置为source.substr(m-1)。

8.

知识点：

  • 字符串处理

程序分析：主要代码来源于第五章的10题。使用了count数组来存储结果，使用之前，需要将数组的元素初始化int count[5] = { 0 };。

9.

知识点：

  • 矩阵转置
  • 多维数组的引用

程序分析：使用引用传递整个数组时，对于多维数组，你需要指定所有维度的大小。

10.

知识点：

  • 头文件 <algorithm>：其中包含了许多与算法相关的功能。该头文件提供了一组通用的算法，这些算法可以用于对容器（如数组、向量、列表等）进行各种操作，包括排序、查找、合并等。以下是一些 <algorithm> 头文件中常用的算法：
    • sort: 对容器进行排序。
    • find: 在容器中查找指定元素。
    • max / min: 返回容器中的最大值或最小值。
    • reverse: 反转容器中的元素顺序。
    • copy: 将一个容器的元素复制到另一个容器中。
    • count: 计算容器中特定元素的个数。
    • accumulate: 对容器中的元素进行累加。

程序分析：先将矩阵中的所有元素放入一个一维数组，进行排序，然后根据题目要求插入原矩阵。

11.

知识点：

  • 头文件 <algorithm>与<vector>

程序分析：将十个字符串储存在向量strs中，然后将strs排序，最后输出。

12.

知识点：

  • 指针数组
  • 字符串处理

程序分析：先将第i个字符串的首地址赋予指针数组p的第i个元素，然后使用冒泡排序将字符串组排序，最后输出。

13.

知识点：

  • 矩形法求定积分

程序分析：该程序采用了左矩形法：

  • 在每个小区间上，用函数在该区间左端点的函数值作为近似值。
  • 将这些近似值相加，乘以小区间的宽度，即可得到对定积分的估计。

读者可能对double (*f)(double)感到陌生，它是一个函数指针的声明。

  • double 表示该函数指针指向的函数的返回类型是 double。
  • (*f) 表示这是一个指针，f 是指针的名称。
  • (double) 表示该指针指向的函数有一个 double 类型的参数。

14.

知识点：

  • 关键字vector

程序分析：使用vector<int> 来存储输入的 n 个数字。函数 reverse 接受一个 const vector<int>& 引用，并逆序输出其中的元素。

15.

知识点：

  • 多个函数的调用
  • 二维数组的处理

程序分析：该题要求实现的功能较多，但都不是很难，一步步实现即可。正所谓难题就是简单题的叠加。

16.

知识点：

  • 数组的处理
  • stoi ： C++ 标准库中的一个函数，用于将字符串转换为整数（int）。它的定义在 <string> 头文件中。

程序分析：该解主要思想在于将数字储存在字符串temp中，当temp 不为空时才将其转换为整数并添加到 digits 中。

17.

知识点：

  • 字符串处理

程序分析：根据strcmp函数原理编写myStrcmp函数，需要注意的是这里为了兼容性，使用了 size_t ，它是用于表示对象大小或元素数量的标准类型。在 C++ 中，许多标准库函数和操作都使用 size_t。因此，在这里使用 size_t 可以保持一致性。

18.

知识点：

  • 指针数组

程序分析：months 是一个指向常量字符串（const char*）的指针数组。输出时，*(months + num -1)可以改为months[num - 1]，数组下标表示法 a[b] 等效于 *(a + b)。

19.

知识点：

  • 指向指针的指针
  • 字符串排序

程序分析：这个程序通过指向指针的指针和指针数组实现了字符串的排序

    • char **p 是指向指针的指针，用于指向字符串数组的首地址。
    • char *pstr[5] 是指针数组，存储了5个指向字符串的指针。
    • char str[5][20] 是二维字符数组，用于存储输入的5个字符串。
  2. 初始化指针数组：
    • pstr[i] = str[i] 将每个指针指向对应的字符数组，即每个 pstr[i] 指向 str[i] 的首地址。

20.

知识点：

  • vector<int*> 作为参数传递

程序分析：使用vector自动处理内存管理，需要注意的是将vector<int*>类型的容器作为参数传递需要写成void sort(vector<int*>& p, int size)如此形式。
